MagentoHosting.pro Development Vagrant Box
 

Usage

  • Install Vagrant
  • Install a provider
    • Parallels Desktop Pro Edition
      • Faster then VirtualBox
      • About $100 per year
      • Only for Mac
      • Install Vagrant plugin:
        vagrant plugin install vagrant-parallels
    • VirtualBox
      • Free
      • For Mac, Linux, Windows
  • Create a working directory and cd into it
  • Install Vagrant hostmanager plugin:
    vagrant plugin install vagrant-hostmanager
  • Get our Vagrantfile:
    curl -O http://vagrant.magehost.pro/Vagrantfile
  • Start Vagrant box:
    vagrant up
    • On the end of the install you will get some information.
    • The local httpdocs is mounted inside the Vagrant box.
    • If you want to connect to the box via SSH:
      vagrant ssh
    • We experienced download errors while testing Vagrant version 1.8.x on Windows. On Mac and Linux it works fine.
